Example for test with https://github.com/taletalesarb/simple-app-with-login

## to run first test

```
python3 -m locust -f locustfile-first-test.py --web-port 9999
```
then head to

http://localhost:9999/

or

http://127.0.0.1:9999/

or

http://0.0.0.0:9999/

```
docker compose run
```
